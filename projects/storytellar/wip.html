StorytellAR (WIP)

StorytellAR will go through another iteration before being published, based on results from the user tests. In the meantime, here are some screenshots, along with a video that was used to present the app to the test participants.

The app was made in Unity, using ARFoundation with ARCore Extensions (for the Recording and Playback API).

A previous version used Firebase (Storage and Realtime Database) to share the created stories between devices. This had to be removed from the current version, as it wouldn't be feasible to test this functionality with the users. It might be present in the final version.

All of the main illustrations and animations belong to project Mobeybou.
